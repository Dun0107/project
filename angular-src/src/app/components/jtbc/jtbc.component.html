JTBC 팀 소개

(팀명 뜻 : Joongbu Teacher 병천 교수님 - JTBC)


팀원 소개



양경모 (팀장)


- 임시 비밀번호 발급 구현
- 발급된 임시 비밀번호 메일로 전송
- 보고서 작성
- 발표자료 제작 및 발표



한수연


- 질문게시판 글 리스트 구현
- 질문게시판 댓글 리스트 구현
- 질문게시판 글보기 구현
- 한줄 마음 전하기 리스트 구현
- 페이지 오류 수정 (로그인 상태 풀리는 오류 등 수정)




박지윤


- 전체 페이지 디자인
- 여러 기능 추가 (학과소개, 과제 제출 기한 확인 등)
- 로그인, 회원가입 구현
- 프로필 구현
- 개인정보 수정 구현
- 질문게시판 글 작성 구현
- 질문게시판 댓글 작성 구현
- 질문게시판 글, 댓글 보기 구현
- 한줄 마음 전하기 글 작성 및 리스트 구현
- 교수님 페이지로 넘어가기 전 인증 구현



송유진


- 과제 제출 구현
- 교수님 페이지에서 제출한 과제 확인 및 다운로드 구현
